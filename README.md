# Cascader Converter 使用说明

### 数据结构转换
将一个List 转换成给前端Cascader组件使用的数据结构。
使用时只需要在对应的getter方法增加注解标明是转换成label, value，还是寻找子节点的parent即可。无需其他额外操作，返回的数据结构支持element ui的cascader组件要求的结构。

等待转换前结构
```java 
public class RegionDTO implements Serializable {
    private static final long serialVersionUID = -5376981511839486821L;

    private String code;

    private String name;

    private String parentCode;

    @Cascader(CascaderFieldEnum.Value)
    public String getCode() {
        return code;
    }

    public void setCode(String code) {
        this.code = code;
    }

    @Cascader(CascaderFieldEnum.Label)
    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }

    @Cascader(CascaderFieldEnum.Parent)
    public String getParentCode() {
        return parentCode;
    }

    public void setParentCode(String parentCode) {
        this.parentCode = parentCode;
    }
}


```
